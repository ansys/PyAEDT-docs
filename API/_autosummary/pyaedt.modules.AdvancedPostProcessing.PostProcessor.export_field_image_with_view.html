export_field_image_with_view#

PostProcessor.export_field_image_with_view(plotName, foldername, exportFilePath, view='isometric', wireframe=True)[source]#

Export a field plot image with a view.

Deprecated since version 0.5.0: Use export_field_jpg() method instead.

Note

For AEDT 2019 R3, this method works only on the ISO view due to a bug in the API. This method works properly in 2021 R1.

Parameters:
plotNamestr

Name of the plot.

exportFilePath

Path for exporting the image file.

viewstr, optional
View to export. Options are "isometric", "xy", "xz", "yz".

The default is "isometric".

wireframebool, optional

Whether to put the objects in the wireframe mode. The default is True.

Returns:
bool

True when successful, False when failed.
